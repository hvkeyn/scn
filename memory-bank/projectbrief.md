# Project Brief: LocalSend

## Обзор проекта

LocalSend - это бесплатное, open-source кроссплатформенное приложение для безопасного обмена файлами и сообщениями между устройствами по локальной сети без необходимости подключения к интернету.

## Основные цели

1. **Безопасный обмен файлами** - передача файлов между устройствами в локальной сети
2. **Кроссплатформенность** - поддержка Windows, macOS, Linux, Android, iOS, Fire OS
3. **Приватность** - работа без внешних серверов, все данные передаются напрямую между устройствами
4. **Простота использования** - интуитивный интерфейс, работа "из коробки"
5. **Высокая скорость** - использование локальной сети для быстрой передачи

## Технические требования

- **Flutter** для UI и кроссплатформенной разработки
- **Rust** для core логики (протокол, криптография, HTTP сервер)
- **HTTPS/TLS** для безопасной передачи данных
- **REST API** для коммуникации между устройствами
- **WebRTC** (опционально) для прямого соединения
- **Multicast DNS** для обнаружения устройств в сети

## Ключевые функции

- Отправка и получение файлов
- Отправка текстовых сообщений
- Обнаружение устройств в локальной сети
- История полученных файлов
- Избранные устройства
- Настройки безопасности (шифрование, порты)
- Поддержка множества языков (106 языков)
- Темная/светлая тема
- Режим портативности (portable mode)
